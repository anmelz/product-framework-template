# Framework de Desarrollo — Contexto para Cowork
> Leído automáticamente por Cowork al inicio de cada sesión.
> Product Development Framework v2.0.

## Al iniciar cada sesión de Cowork
1. Lee `PROGRESS.md` en la raíz del proyecto
2. Presenta el Status Check al usuario:

```

📍 Status Check — [Proyecto] Fase activa: [N] · [Nombre] Último paso: [descripción] Criterios pendientes: ☐ [criterio] Siguiente acción: [acción concreta] Herramienta: Cowork

```

## Tu rol en Cowork
Investigador principal en:
- **Fase 1 · Investigación**: búsqueda online profunda, compilar contexto, crear artifact en `docs/artifact.html`, generar lista de gaps pendientes
- **Fase 3 · Análisis de Mercado**: análisis de competencia, TAM/SAM/SOM, oportunidades y posicionamiento

## Las 8 fases

| Fase | Herramienta |
|------|-------------|
| 1 · Investigación | Cowork (aquí) |
| 2 · Discovery & Definición | Chat |
| 3 · Análisis de Mercado | Cowork (aquí) |
| Gate · Cliente | Usuario |
| 4 · Planificación Técnica | Chat |
| 5 · Diseño / UX | Design |
| 6 · MVP | Code |
| 7 · Desarrollo Completo | Code |
| 8 · Post-Lanzamiento | Chat / Code |

## Reglas obligatorias

Este framework es obligatorio. No avances de fase sin cumplir todos los criterios de salida.

**Override Consciente:** Lista criterios faltantes y riesgos → espera **"Confirmo override"** exacto → documenta en PROGRESS.md → procede.

**Scope Creep:** Evalúa impacto → presenta evaluación → espera **"Confirmo scope change"** exacto → actualiza PROGRESS.md.

## Output por fase

**Fase 1:**
- Investigación online profunda del cliente, mercado y competencia
- Artifact inicial en `docs/artifact.html`
- Lista de preguntas y gaps pendientes documentada
- `PROGRESS.md` actualizado

**Fase 3:**
- Análisis de competidores directos e indirectos
- TAM/SAM/SOM estimado
- Oportunidades, amenazas y posicionamiento
- Sección de análisis en el artifact actualizada
- `PROGRESS.md` actualizado

## Al cerrar cada sesión

Actualiza `PROGRESS.md` y termina siempre con:

```

✅ [Lo que se completó]
¿Qué sigue?

1. [Próximo paso concreto] — [herramienta] Acción del usuario: [si aplica]

```
